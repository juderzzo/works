Basics

In order to plug something into mathematica you must first create a cell. In this cell - which by default will be set to the wolfram language, you can hold shift and enter to return a result

The % symbol refers to the most recent output. Wolfram mathematica has a plurality of built in functions! We can either memorize them, ore even better, look them up on wolfram itself.

Lists in Mathematic represent collections, they are represented like such {...}

In the wolfram language, you don’t only use decimals, you can also use fractions, by using control, and evaluating them with the N command

Variables can be declared traditionally, or in an expression, they can include numbers. We represent the multiplication between 2 variables by using a space.

Functions can also be declared using the f[x_] := command and defining a function

Algebra

Mathematica is an extremely powerful tool when it comes to solving algebra. It can solve, or approximate the solution to any polynomial with the roots command, as well as factor polynomials.

It can easily accommodate complex roots, and approximate all roots with NSolve and NRoots. It can even handle polynomials of degree 420!

In order to plot something, all you have to do is use the Plot or RegionPlot command (Also, to reduce an inequality, use the reduce command to reduce nay inequality), but you have to use the show command, and put all your plots in a list

Geometry

Use the Graphics command before every shape, using the rectangle and disk command, for example.

You can venerate triangles with the side angle side command, as well as the ASA. It will return the coordinates, which, if you plug into graphics, will display the triangle You can take areas, draw 3D sections, use the ball command, and on top of that look up volumes yourself. You can also rotate points with a rotate command.

You can also use your trig functions (by default in radians, but you can use the degree symbol for degrees. You can also expand and factor trigonometric expressions like in Algebra 2
